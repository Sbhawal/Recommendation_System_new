About

A content based Recommendation System made by ranking the listings using cosine simmilarity. The features used for ranking are primarily Genre and Show Summary with year of release, votes and Rating. Web Output is presented from local server, on absence of any relevant data IMDb from RapidAPI is used. Codes present in Github repository.
